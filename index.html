NodeJS = Node

--- In this talk we're going to discuss the increasingly popular Node.JS application platform. From here on out I'm going to refer to Node.JS simply as "Node", with a capital 'N'.

So, what is Node?

  • "Node.js is an open source, cross-platform runtime environment for server-side and networking applications". - Wikipedia
  • Also an application and platform for building application - for use on and off the net
  • An application optimized for performing high speed input/output operations
--- Node is software which allows programmers to build applications, using JavaScript, for use on and off the net. It's often thought of as a tool for building backend server applications, however, it's actually much more than that. The Node software essentially hosts a JavaScript execution engine and provides built-in functionality for performing highly optimized input/output operations. These I/O operations cover things like reading and writing files, and network operations such as accepting connections and making remote requests. Using JavaScript and Node's I/O features allows developers to build applications which can interact with the current machine and its local resources, while allowing communication between hosted applications and networked resources. We'll see further examples of Node in action later in this presentation.
